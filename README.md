# Find-the-Pairs
Find Pairs in a BST

Write a program to find all the pairs of values from a given BST, where the sum of the pair is equaled
to a given value.

Example Output:     
Enter the number of nodes in the BST: 10     
Enter the data: 5 4 6 3 7 9 10 8 2 1    
Enter a value: 8    
Pairs: (5, 3) (6, 2) (7, 1)
